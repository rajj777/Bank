===================================== Create certificates rootca,sub,private =====================================

Root CA Certificate

· Generate root CA private key:
·= openssl genpkey -algorithm RSA -out root.key

· Generate root CA Certificate Signing Request (CSR):
·= openssl req -new -key root.key -out root.csr

· Self-sign the Root CA certificate:
= openssl x509 -req -in root.csr -signkey root.key -out root.crt -days 3650

· View the file in a readable format:
·= openssl x509 -in root.crt -text -noout
-----------------------

Sub CA Certificate

· Generate sub CA private key:
·= openssl genpkey -algorithm RSA -out sub.key

· Generate sub CA Certificate Signing Request (CSR):
·= openssl req -new -key sub.key -out sub.csr

. Sign the sub CA CSR with Root CA:
.= openssl x509 -req -in sub.csr -CA root.crt -CAkey root.key -CAcreateserial -out sub.crt -days 1825

· View the file in a readable format:
.= openssl x509 -in sub.crt -text -noout


---------------------

  Issue Digital Certificates from Sub CA
· Generate some server's private key:
·= openssl genpkey -algorithm RSA -out server.key

· Generate CSR for an end entity (e.g. some server): cdac.ac.in
